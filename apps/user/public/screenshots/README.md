# PWA Screenshots

Place your app screenshots in this directory for the app store listing:

## Required Screenshots

- `home.png` - Wide format (1280x720) - Desktop/tablet view
- `mobile.png` - Narrow format (750x1334) - Mobile view

## Screenshot Guidelines

- **Format**: PNG or JPEG
- **Content**: Show your app's key features
- **Quality**: High resolution, clear UI elements
- **Wide Format**: Landscape orientation (desktop/tablet)
- **Narrow Format**: Portrait orientation (mobile)

## Recommended Sizes

### Wide (Desktop/Tablet)
- 1280x720 pixels
- 2560x1440 pixels (Retina)

### Narrow (Mobile)
- 750x1334 pixels (iPhone)
- 1080x1920 pixels (Android)
- 1170x2532 pixels (iPhone Pro Max)

## Tips

1. Use real app content (not placeholder data)
2. Show the most compelling features
3. Ensure good contrast and readability
4. Include at least 2-3 screenshots for each form factor
5. Screenshots appear in app installation prompts on some browsers

## Optional: More Screenshots

You can add more screenshots by updating the manifest.webmanifest file.
